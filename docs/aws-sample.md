# AWS 基礎用語（サンプル基準知識）

これは動作確認用の最小サンプルです。実運用では AWS 公式ドキュメントを取り込んでください。

## Amazon S3

Amazon S3 はオブジェクトストレージサービスです。ファイルをオブジェクトとしてバケットに保存します。OS を入れて起動するようなブロックデバイスではありません。高い耐久性（99.999999999%、イレブンナイン）を持ち、大量の画像・動画・バックアップ・静的コンテンツの保存に向いています。

## Amazon EBS

Amazon EBS はブロックストレージサービスです。EC2 インスタンスにアタッチして、OS やデータベースのディスクとして使います。単一のアベイラビリティゾーン内に存在し、スナップショットを S3 に保存できます。S3 とは異なり、EC2 に接続して使うディスクです。

## Amazon EC2

Amazon EC2 はクラウド上の仮想サーバー（インスタンス）を提供するコンピューティングサービスです。OS を選んで起動し、アプリケーションを実行します。ストレージとして EBS をアタッチできます。

## 可用性と耐久性

耐久性（Durability）は、保存したデータが失われずに残り続ける度合いです。可用性（Availability）は、必要なときにサービスやデータにアクセスできる度合いです。S3 は高い耐久性と高い可用性の両方を提供しますが、両者は別の概念です。

## IAM

AWS IAM は認証と認可を管理するサービスです。ユーザー・グループ・ロール・ポリシーで、誰が何のリソースにアクセスできるかを制御します。IAM ロールは一時的な権限を付与する仕組みで、EC2 などのサービスに割り当てられます。
